Instructions
  1. First check if jquery is included, if NOT add these files to the <head>
    • <script type="text/javascript" src="js/jquery–1.11.1.min.js"></script>
    • <script type="text/javascript" src="js/ninja.contact.js"></script>
    • <script src="https://www.google.com/recaptcha/api.js" async defer></script>
    If jquery is already included then add these lines below where jquery is included:
    • <script type="text/javascript" src="js/ninja.contact.js"></script>
    • <script src="https://www.google.com/recaptcha/api.js" async defer></script>
  2. Give the <form> the following attributes:
    • name="frmContactUs"
    • id="frmContactUs"
    • method="POST"
    • action="submit.contact.php"
  3. For all the inputs strictly use these names and you must use a placeholder attribute. (Whatever you put in the placeholder attribute will be the description for that field in the email.):
    • Name input: name="name" placeholder="Anything you want...(eg Full Name)"
    • Phone Number input: name="contactus" placeholder="Anything you want...(eg Phone No)"
    • Email input: name="email" placeholder="Anything you want...(eg E-mail Address)"
    • Message input: name="message" placeholder="Anything you want...(eg Your query)"
    • For any custom input:
      name="custom_input[]" placeholder="Anything you want...(eg Reference No)"
    • For any custom dropdown:
      name="custom_select[]" placeholder="Anything you want...(eg Favorite Color)"
    • For any custom textarea:
      name="custom_textarea[]" placeholder="Anything you want...(eg Other Comments)"
    • If you want to use the captcha code in the form simply add the following <div>:
      <div class="g-recaptcha"></div >
    • Make sure that the submit element has the following id attribute:
      id="btnSubmit"
  4. Add the following <div> at the end of the form for replies:
    <div id="reply"></div >
  5. Other Notes:
    • if you use a dropdown make sure the first <option>:
      <option value=""">–– select ––</option>
    • Disable any original form validation
